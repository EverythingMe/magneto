Run the demo
============

Install magento::

  $ pip install magneto

Enter a new folder::

  $ mkdir calc
  $ cd calc

Init the calc demo::

  $ magneto init calc

Run the tests::

  $ magneto run tests/

This will install the app on your device, run the tests and display a final report.
